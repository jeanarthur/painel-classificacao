# Painel de Classificação

# Instalação

Clone o repositório \
`git clone https://github.com/jeanarthur/painel-classificacao.git`

Instale as dependências do Node \
`npm install`

# Execução

## Página

Abra o HTML no navegador ou instale a extensão do LiveServer no Visual Code.


## Javascript (NodeJS)

Navegue até a pasta \
`cd src/`

Execute \
`node funcoes.js`
